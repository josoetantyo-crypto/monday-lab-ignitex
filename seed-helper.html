Migrasi data dari dashboard lama ke Supabase. Jalankan sekali saja.

Cara ambil data dari dashboard lama:
1. Buka file dashboard.html di browser
2. Tekan F12 → tab Console
3. Ketik perintah ini lalu Enter:
   copy(JSON.stringify(sessions))
4. Paste hasilnya di kotak "Data JSON" di bawah
